测试(Testing)¶

单元测试(Unit test)¶

查看 building Hyperledger Fabric 获得单元测试命令。

查看 单元测试覆盖报告

为了查看一个包或者所有子包的覆盖率，替换 -cover 执行单元测试

go test ./... -cover

为了查看一个包中哪一行没有被覆盖，用源代码生成一个带有覆盖率注释的html报告

go test -coverprofile=coverage.out
go tool cover -html=coverage.out -o coverage.html
